# 一封代码情诗

![logo](./logo.png)

## Issues投稿形式

  用任意编程语言，选择以下 2 个方向之一写下不超过 50 行代码

  * **代码写情诗**
  * **代码写决绝诗**

### 示例

![example](./example.png)

```
  此生爱你 只增不减 - 掘金@March21Sunny
```

以 Carbon 生成对应的代码图片，在线代码图片生成器 Carbon 传送门：[👀这里](https://carbon.now.sh)。
